# google-workspace

Google Workspace operations via `gog` CLI — Gmail, Calendar, Drive, Docs, Sheets, Slides, Forms, Contacts, Tasks, Admin, Chat, YouTube, Maps, Photos, Meet.

## Files

- `SKILL.md` — Skill definition with execution steps and constraints
- `references/setup.md` — First-time owner setup guide
- `references/auth-flow.md` — Headless OAuth, re-authorization, security
- `references/account-resolution.md` — Multi-account selection rules
- `references/commands.md` — Intent → gog command mapping
- `references/examples.md` — Example conversations with permission checks

## Permissions

- `read` — Read-only operations (scoped per service)
- `write` — Mutations (scoped per service)
